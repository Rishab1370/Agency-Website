# NeuralCraft — AI-Powered Digital Agency

![NeuralCraft Banner](https://images.unsplash.com/photo-1677442136019-21780ecad995?auto=format&fit=crop&q=80&w=1200)

**NeuralCraft** is a premium, full-service digital and AI agency landing page and multi-page platform. Built with a focus on high performance, modern aesthetics (glassmorphism), and seamless user experience, it showcases the intersection of cutting-edge AI transformation and elite digital craftsmanship.

---

## 🚀 Live Demo
Check out the live repository and code: [GitHub Repo](https://github.com/Rishab1370/Agency-Website)

---

## ✨ Features

- **Multi-Page Architecture**: Fully implemented Home, About Us, Case Studies, Blog, and Careers pages.
- **Premium Design System**: Sophisticated dark-mode UI utilizing glassmorphic components, subtle gradients, and fluid typography.
- **Dynamic Animations**: Powered by `framer-motion` for staggered entrances, interactive hover states, and smooth section transitions.
- **High Performance**: Optimized using Next.js 15, ensuring instantaneous route transitions and minimal bundle size.
- **Global Navigation**: Persistent Navbar and Footer optimized for cross-page jumping and section-anchoring.
- **Consultation Modal**: Integrated premium modal system for lead generation and client inquiries.
- **SEO Ready**: Full metadata configuration, structured semantics, and optimized font loading for maximum search visibility.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Context Management**: React Context API (Theme, Modal)
- **Deployment**: [Vercel](https://vercel.com/) (Recommended)

## 📦 Getting Started

### Prerequisites
- Node.js 18+
- npm or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rishab1370/Agency-Website.git
   cd Agency-Website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

---

## 📁 Folder Structure

- `app/`: Next.js App Router directory (Routes/Pages)
- `components/`: Reusable UI components and page sections
- `lib/`: Data models, utility functions, and Context providers
- `public/`: Static assets (Images, Fonts)
